What are CSS Variables?
CSS Variables let you store values (like colors) and reuse them everywhere. This makes your code cleaner and easier to update.

üìù Properly Formatted Code:

/* Define your color palette */
:root {
    /* Primary Colors */
    --primary: #2563eb;
    --primary-light: #3b82f6;
    --secondary: #8b5cf6;
    --accent: #f59e0b;
    
    /* Text Colors */
    --text-white: #ffffff;
    --text-gray: #94a3b8;
}

/* Using the variables */
.button {
    background: var(--primary);
    color: var(--text-white);
    padding: 15px 30px;
    border-radius: 50px;
}

.button:hover {
    background: var(--primary-light);
}

üîç Line-by-Line Explanation:

Line 1-2: We define our root variables inside :root
Line 4-7: Each color gets a descriptive name like --primary
Line 13: We use the variable with var(--primary)
Line 19: On hover, we change to the lighter version
üí° Why This is Better: Instead of typing #2563eb 50 times, you define it once and use var(--primary). If you want to change the color later, you only update one line!

3Ô∏è‚É£ CSS Animations - Make Things Move!

What are Keyframe Animations?
Animations let you move, rotate, scale, or change elements over time. They make your website feel alive!

üìù Step 1: Define the Animation:

/* Define what the animation does */
@keyframes fadeInUp {
    /* Starting state (0% or "from") */
    from {
        opacity: 0;                    /* Invisible */
        transform: translateY(50px);   /* 50px below */
    }
    
    /* Ending state (100% or "to") */
    to {
        opacity: 1;                    /* Fully visible */
        transform: translateY(0);      /* Original position */
    }
}

üìù Step 2: Apply the Animation:

.my-box {
    animation: fadeInUp 1s ease-out;
}
üí° Common Timing Functions:
‚Ä¢ ease - Slow start, fast middle, slow end
‚Ä¢ ease-in - Slow start
‚Ä¢ ease-out - Slow end
‚Ä¢ linear - Constant speed
