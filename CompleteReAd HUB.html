Advanced CSS Transforms & 3D Effects

ðŸŽ­ Understanding CSS Transforms

Transforms allow you to move, rotate, scale, and skew elements without affecting document flow. They're hardware-accelerated, making them perfect for animations.

Why use transforms? They're 60fps smooth and don't cause reflows (page re-calculations).

/* 2D Transforms */ /* Translate - Move elements */ .translate-example { transform: translateX(50px); /* Move right 50px */ transform: translateY(-30px); /* Move up 30px */ transform: translate(50px, -30px); /* Move right 50px, up 30px */ } /* Scale - Resize elements */ .scale-example { transform: scale(1.5); /* 150% size */ transform: scale(0.8); /* 80% size */ transform: scaleX(2); /* Double width only */ transform: scale(1.2, 0.8); /* Width 120%, height 80% */ } /* Rotate - Spin elements */ .rotate-example { transform: rotate(45deg); /* Clockwise 45 degrees */ transform: rotate(-90deg); /* Counter-clockwise 90 degrees */ } /* Skew - Slant elements */ .skew-example { transform: skewX(20deg); /* Slant horizontally */ transform: skewY(10deg); /* Slant vertically */ } /* Combining Multiple Transforms */ .combined { transform: translate(50px, 50px) rotate(45deg) scale(1.2); /* Order matters! Read right-to-left */ } /* 3D Transforms */ /* Enable 3D Space */ .container-3d { perspective: 1000px; /* Depth perception (distance from viewer) */ perspective-origin: center; /* Vanishing point */ } /* 3D Rotate */ .rotate-3d { transform: rotateX(45deg); /* Rotate around X-axis */ transform: rotateY(45deg); /* Rotate around Y-axis */ transform: rotateZ(45deg); /* Rotate around Z-axis (same as rotate()) */ transform-style: preserve-3d; /* Children maintain 3D position */ } /* 3D Translate */ .translate-3d { transform: translateZ(100px); /* Move towards viewer */ transform: translate3d(50px, 50px, 100px); /* X, Y, Z */ } /* Flip Card Effect (Real-world Example) */ .flip-card { width: 300px; height: 200px; perspective: 1000px; } .flip-card-inner { position: relative; width: 100%; height: 100%; transition: transform 0.6s; transform-style: preserve-3d; } .flip-card:hover .flip-card-inner { transform: rotateY(180deg); } .flip-card-front, .flip-card-back { position: absolute; width: 100%; height: 100%; backface-visibility: hidden; /* Hide back side */ } .flip-card-back { transform: rotateY(180deg); } /* 3D Cube Example */ .cube-container { width: 200px; height: 200px; perspective: 1000px; } .cube { width: 100%; height: 100%; position: relative; transform-style: preserve-3d; animation: rotateCube 10s infinite linear; } .cube-face { position: absolute; width: 200px; height: 200px; border: 2px solid #667eea; background: rgba(102, 126, 234, 0.1); backdrop-filter: blur(10px); } .cube-face.front { transform: rotateY(0deg) translateZ(100px); } .cube-face.back { transform: rotateY(180deg) translateZ(100px); } .cube-face.left { transform: rotateY(-90deg) translateZ(100px); } .cube-face.right { transform: rotateY(90deg) translateZ(100px); } .cube-face.top { transform: rotateX(90deg) translateZ(100px); } .cube-face.bottom { transform: rotateX(-90deg) translateZ(100px); } @keyframes rotateCube { to { transform: rotateX(360deg) rotateY(360deg); } }

LocalStorage & Data Persistence

ðŸ’¾ What is LocalStorage?

LocalStorage is a way to store data in the user's browser that persists even after they close the page. Think of it as a small database that lives in the browser.

Capacity: ~5-10MB per domain (varies by browser)

Data Type: Stores only strings (we'll learn to work around this)

// Basic LocalStorage Operations // 1. STORING DATA (setItem) localStorage.setItem('username', 'JohnDoe'); localStorage.setItem('theme', 'dark'); localStorage.setItem('fontSize', '16'); // 2. RETRIEVING DATA (getItem) const username = localStorage.getItem('username'); console.log(username); // Output: "JohnDoe" // 3. REMOVING DATA (removeItem) localStorage.removeItem('fontSize'); // 4. CLEARING ALL DATA (clear) localStorage.clear(); // Deletes everything! // 5. CHECKING IF KEY EXISTS if (localStorage.getItem('username') !== null) { console.log('Username exists!'); } /* WORKING WITH OBJECTS (Must stringify!) */ // Storing an object const user = { name: 'John Doe', email: 'john@example.com', age: 25, preferences: { theme: 'dark', language: 'en' } }; // Convert object to JSON string localStorage.setItem('user', JSON.stringify(user)); // Retrieve and parse back to object const retrievedUser = JSON.parse(localStorage.getItem('user')); console.log(retrievedUser.name); // Output: "John Doe" /* STORING ARRAYS */ const todoList = [ { id: 1, text: 'Learn JavaScript', completed: false }, { id: 2, text: 'Build a project', completed: true } ]; // Store array localStorage.setItem('todos', JSON.stringify(todoList)); // Retrieve array const todos = JSON.parse(localStorage.getItem('todos')) || []; /* REAL-WORLD EXAMPLE: Theme Switcher */ function setTheme(theme) { // Save to localStorage localStorage.setItem('theme', theme); // Apply to page document.body.className = theme; } function loadTheme() { // Load saved theme or default to 'light' const savedTheme = localStorage.getItem('theme') || 'light'; document.body.className = savedTheme; } // Load theme when page loads window.addEventListener('DOMContentLoaded', loadTheme); /* REAL-WORLD EXAMPLE: Shopping Cart */ class ShoppingCart { constructor() { this.items = this.load(); } load() { const cartData = localStorage.getItem('cart'); return cartData ? JSON.parse(cartData) : []; } save() { localStorage.setItem('cart', JSON.stringify(this.items)); } addItem(product) { this.items.push(product); this.save(); } removeItem(productId) { this.items = this.items.filter(item => item.id !== productId); this.save(); } clear() { this.items = []; localStorage.removeItem('cart'); } } // Usage const cart = new ShoppingCart(); cart.addItem({ id: 1, name: 'Laptop', price: 999 }); /* REAL-WORLD EXAMPLE: Auto-save Form */ const form = document.getElementById('contactForm'); const inputs = form.querySelectorAll('input, textarea'); // Auto-save as user types inputs.forEach(input => { // Load saved value const savedValue = localStorage.getItem(`form_${input.name}`); if (savedValue) { input.value = savedValue; } // Save on input change input.addEventListener('input', () => { localStorage.setItem(`form_${input.name}`, input.value); }); }); // Clear saved data on successful submit form.addEventListener('submit', (e) => { e.preventDefault(); // Submit form... // Clear saved data inputs.forEach(input => { localStorage.removeItem(`form_${input.name}`); }); }); /* ERROR HANDLING */ function safeSetItem(key, value) { try { localStorage.setItem(key, value); return true; } catch (e) { if (e.name === 'QuotaExceededError') { console.error('LocalStorage is full!'); } return false; } }

ðŸ”‘ LocalStorage Key Concepts:

  • Persistent: Data survives browser restarts
  • Synchronous: Blocks code execution (use carefully)
  • String Only: Must use JSON.stringify() for objects
  • Domain Specific: Each domain has its own storage
  • No Expiration: Data stays until manually deleted
  • Size Limit: ~5-10MB per domain
âš ï¸ Security Warnings:
  • Never store passwords or sensitive data
  • LocalStorage is not encrypted
  • Vulnerable to XSS attacks
  • Use sessionStorage for temporary data
  • Consider IndexedDB for larger data

    // 3. Prepare the data to send
    const requestBody = {
        contents: [{
            parts: [{
                text: message
            }]
        }]
    };
    
    // 4. Build the API URL
    const apiUrl = 
        'https://generativelanguage.googleapis.com/' +
        'v1beta/models/gemini-pro:generateContent' +
        '?key=' + apiKey;
    
    try {
        // 5. Send request to API (wait for response)
        const response = await fetch(apiUrl, {
            method: 'POST',
            headers: {
                'Content-Type': 'application/json'
            },
            body: JSON.stringify(requestBody)
        });
        
        // 6. Convert response to JSON
        const data = await response.json();
        
        // 7. Get AI's answer from response
        const aiAnswer = data
            .candidates[0]
            .content
            .parts[0]
            .text;
        
        // 8. Show AI's answer on screen
        displayMessage(aiAnswer);
        
    } catch (error) {
        // 9. If something goes wrong, show error
        console.error('Error:', error);
        alert('Failed to connect to AI');
    }
}
